# sort

Data sorting exercise. Using one collection sort strategy to order the list of persons using the name and the age. Example.
- Ana 18
- Camilo 17
- Camilo 19
- Juan 20
- Miguel 18
- Miguel 23


## Tools

- java 11
- junit 5.4.0
- maven


## Class and Interfaces

### Person
Class Person with the name and the age.
